## 👨‍💻 Sobre este Repositório

Este repositório reúne os projetos, desafios, sprints, Global Solutions e atividades práticas desenvolvidos ao longo da Graduação ON em Inteligência Artificial da FIAP.

Durante a formação, foram aplicados conceitos de Inteligência Artificial, Machine Learning, Ciência de Dados, Engenharia de Software, Internet das Coisas (IoT), Banco de Dados e Computação em Nuvem na construção de soluções voltadas para problemas reais.

Além de registrar a evolução acadêmica, este repositório funciona como um portfólio técnico, demonstrando competências adquiridas na concepção, desenvolvimento e implementação de sistemas inteligentes orientados por dados.

---

## 🎯 Objetivos

Este repositório foi criado com os seguintes propósitos:

* Organizar todas as entregas realizadas durante o curso;
* Registrar a evolução técnica e profissional dos integrantes;
* Consolidar conhecimentos adquiridos em diferentes disciplinas;
* Aplicar metodologias ágeis e boas práticas de desenvolvimento;
* Construir um portfólio acadêmico e profissional estruturado;
* Demonstrar a aplicação prática de Inteligência Artificial em cenários reais.

---

## 🚀 Principais Projetos Desenvolvidos

### 🌱 FarmTech Solutions

Projeto multidisciplinar desenvolvido ao longo de diversas fases do curso com foco na transformação digital do agronegócio.

Principais funcionalidades:

* Monitoramento agrícola inteligente;
* Sistema de análise BRIX para cana-de-açúcar;
* Irrigação inteligente baseada em sensores;
* Dashboards analíticos;
* Banco de dados Oracle;
* Modelos preditivos para apoio à tomada de decisão.

---

### 📊 Analytics e Banco de Dados

Projetos voltados para:

* Modelagem de dados;
* SQL avançado;
* Oracle Database;
* Procedures e Views;
* Integração de dados;
* Construção de dashboards e indicadores.

---

### 🤖 Machine Learning

Desenvolvimento de modelos para:

* Predição de produtividade agrícola;
* Classificação de dados;
* Clusterização;
* Identificação de padrões;
* Avaliação de desempenho de algoritmos.

Técnicas utilizadas:

* Regressão Linear;
* Árvores de Decisão;
* Random Forest;
* Modelos Ensemble;
* Aprendizado Não Supervisionado.

---

### 👁️ Visão Computacional

Aplicação de técnicas de processamento de imagens para:

* Reconhecimento visual;
* Monitoramento automatizado;
* Detecção de padrões;
* Apoio à tomada de decisão.

Tecnologias utilizadas:

* OpenCV;
* Python;
* Machine Learning.

---

### 🌎 Global Solution – HumanIza

Projeto desenvolvido para apoiar integração profissional e orientação de carreira por meio de Inteligência Artificial.

Recursos:

* Assistente virtual;
* Recomendação personalizada;
* Processamento de linguagem natural;
* Interface interativa.

---

### ☀️ Global Solution – HeliOS

Sistema inteligente para monitoramento e previsão de tempestades solares.

Objetivos:

* Análise de eventos espaciais;
* Avaliação de riscos;
* Modelos preditivos;
* Visualização de dados.

---

### 🎭 Challenge – Aurora Totem

Desenvolvimento de MVP para interação inteligente entre usuários e plataforma.

Componentes:

* Backend FastAPI;
* Dashboard Streamlit;
* Integração com banco Oracle;
* Simulação IoT;
* Inteligência Artificial aplicada.
